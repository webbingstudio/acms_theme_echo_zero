## echo_zeroとは

「echo_zero」は、a-blog cms専用のテーマです。a-blog cmsの多彩な機能のうち、業務で特に採用される機能のみを選んで実装しています。

デモサイト  
[http://cms-skill.com/echo_zero_demo/](http://cms-skill.com/echo_zero_demo/)

リリースノート

echoシリーズ ポータルサイト
